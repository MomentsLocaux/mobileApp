# P2 Post-MVP Proche

## Améliorations Utiles Mais Non Bloquantes

### MVP-P2-001 - Analytics produit minimal

Mesurer activation, recherche, vue événement, création, favori, check-in, report, suppression compte.

### MVP-P2-002 - Accessibilité approfondie

VoiceOver/TalkBack, touch targets, contrastes, font scaling.

### MVP-P2-003 - Offline partiel

Brouillon local, retry upload, cache minimal.

### MVP-P2-004 - Performance fine

Cache carte, thumbnails, optimisation stats, mesure appareils réels.

### MVP-P2-005 - Notifications backend/push

Déplacer création notifications backend et préparer push après inbox stable.

### MVP-P2-006 - Anti-spam avancé

Quotas/rate limits, trust score, journalisation renforcée.

### MVP-P2-007 - Design system secondaire

Polish badges, empty/loading/error states, cards et formulaire.

### MVP-P2-008 - Automatisation data/staging

Purge orphelins, seed automatisé, fixtures e2e.
